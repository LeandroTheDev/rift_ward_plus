# Rift Ward Plus
Add three new items, one for detecting rift wards, other for setting any rift ward and view the distance, another is for detecting the rift ward pinger

# Building
- Install .NET in your system, open terminal type: ``dotnet new install VintageStory.Mod.Templates``
- Create a template with the name ``LevelUP``: ``dotnet new vsmod --AddSolutionFile -o RiftWardPlus``
- [Clone the repository](https://github.com/LeandroTheDev/rift_ward_plus/archive/refs/heads/main.zip)
- Copy the ``CakeBuild`` and ``build.ps1`` or ``build.sh`` and paste inside the repository

Now you can build using the ``build.ps1`` or ``build.sh`` file
